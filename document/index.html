CP-ACE functions

Ryutaro Susukita1

CP-ACE functions exchange data between processes where we know only to which processes each process sends data but do not know from how many processes each process receives in parallel programming. Unfortunately, we cannot write simply such data exchange using Message Passing Interface (MPI) Version 3 [1]. If we use CP-ACE functions, we can exchange data using a single communication function with high efficiency. CP-ACE functions are implemented on MPI or Advanced Communication Primitives (ACP) [2]. The functions are written in C language, but the function on MPI has Fortran interfaces.
